# 📊 Financial News Sentiment Analysis

A **Multiclass NLP Classification System** that analyzes financial news and predicts sentiment using Machine Learning.

you can pull docker image with 

docker pull kumar2700/financialnewssentimentanalysis:latest

and run with 

docker run -p 8000:8000 -p 8501:8501 kumar2700/financialnewssentimentanalysis:latest

Built with:

* ⚡ FastAPI (Backend API)
* 🎯 Streamlit (Frontend UI)
* 🐳 Docker (Containerization)



## 🚀 Features

* Multiclass sentiment prediction (e.g., Positive, Negative, Neutral)
* Text preprocessing with stemming & stopword removal
* TF-IDF vectorization + feature selection
* Multiple ML models tested (Naive Bayes, KNN, SVM, XGBoost)
* Best model: **Support Vector Classifier (SVC)**
* Interactive UI using Streamlit
* REST API using FastAPI
* Dockerized for easy deployment
